一台电脑使用两个Github账号

2018-10-27

github

起因，主要是作者本人想在github上面部署两个静态博客，可是一个账号只能部署一个，所以只好申请两个账号。

思路

管理两个 SHH key

解决方案

生成两个SSH key

为了举例方便，这里使用“one”和“two”两个账户。下同。

不要一路回车，分别在第一个对话框的时候输入重命名（id_rsa_one和id_rsa_two），这样会生成：

id_rsa_one
id_rsa_one.pub

id_rsa_two
id_rsa_two.pub

两份包含私钥和公约的4个文件。

ps：ssh-keygen是linux命令，可以让两个机器之间使用ssh而不需要用户名和密码

一定要在~/.ssh路径下运行命令行，不然生成的文件不会出现在当前目录
